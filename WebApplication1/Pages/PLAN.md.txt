# План разработки приложения "Коммерческое предложение"

1. **Предпросмотр на веб-странице**
   - Данные пользователя после отправки формы отображаются внизу страницы (без PDF).
   - Кнопка "Скачать PDF" доступна только после предпросмотра.

2. **Формирование PDF**
   - Подключена библиотека QuestPDF.
   - Генерация документа по введённым данным.

3. **Стилизация веб-страницы**
   - Добавить CSS для красивого отображения формы и предпросмотра.

4. **Хранение данных**
   - Для начала — в памяти (список объектов ProposalRequest).
   - В перспективе — подключить БД (SQL).

5. **Расширение функционала**
   - Добавить новые опции в форму (планировка, 3D-визуализация и т.д.).
   - Автоматический расчёт стоимости.
